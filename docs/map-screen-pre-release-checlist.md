# Map Screen – Pre-Release Checklist

## Summary
This checklist ensures the Map Screen is fully functional and reliable before release. It validates core workflows including prospect and customer management, follow-ups, objections, trip logging, property acquisition, toolbar actions, and scorecard accuracy to guarantee a smooth field experience.

---

## General

1. Can the prospect popup display open correctly?
2. Can the customer popup display open correctly?
3. Can a prospect be disqualified, and does the marker change to an “X”?
4. Can a knock be logged for a disqualified prospect?
5. Can a prospect be re-qualified?
6. Can a follow-up be scheduled for a prospect?
7. Do notes automatically log for a follow-up?
8. Can an objection be added during the follow-up form?
9. Can a trip be logged during the follow-up form?
10. If a prospect with follow-ups is deleted, are its follow-ups also deleted?
11. Can a prospect be converted into a customer?
12. Can a customer be converted back into a prospect?
13. Can a follow-up be scheduled for a customer?
14. Can an objection be added while scheduling a follow-up for a customer?
15. Can a note be logged while scheduling a follow-up for a customer?
16. Can a trip be logged while scheduling a follow-up for a customer?
17. Does deleting a customer remove all of its corresponding follow-ups?

---

## New Property Acquisition Workflow

1. Can a new prospect be added to the map?
2. Can a newly added prospect be removed from the map?
3. Can a new customer be added to the map?
4. Can a newly added customer be removed from the map?

---

## Bulk Property Acquisition Workflow

1. Can multiple properties be added at once?
2. Can all properties in a bulk acquisition be added without duplicates?
3. Can only a subset of properties be selected and added from a bulk acquisition?

---

## Toolbar Workflow #1 – Location

1. Does selecting the location finder center the map on the user’s location?
2. Does rotating the phone update the direction cone correctly?

---

## Toolbar Workflow #2 – QR Code

1. Does clicking the QR button open the QR code generator?
2. Does changing the link update the QR code correctly?
3. Does scanning the QR code navigate to the correct page?

---

## Toolbar Workflow #3 – Search

1. Does clicking the search button expand the search field?
2. Does entering a partial address usually navigate to the correct location?
3. Does selecting a suggested address navigate to the correct place?
4. If a searched address already has a marker, does it open the prospect popup?
5. If a searched address has no marker, does it start the new property acquisition workflow?

---

## Scorecards

1. Does clicking each scorecard open a detented popup sheet?
2. Does the knocking scorecard update correctly after logging multiple knocks?
3. Does the sales scorecard update correctly after logging multiple sales?
